## Third-Party Libraries

### Quantum-Resistant Cryptography
- **liboqs** (Open Quantum Safe) - CRYSTALS-Dilithium and CRYSTALS-Kyber implementations
  - ML-DSA-87 (Dilithium5) - NIST FIPS 204
  - ML-KEM-1024 (Kyber1024) - NIST FIPS 203

### Database
- **RocksDB** - Persistent blockchain storage with Zstandard compression

### GUI
- **Qt5** - Cross-platform graphical user interface

### Cryptography
- **OpenSSL** - SHA-256 and general cryptographic functions

### Build System
- **CMake** - Cross-platform build configuration
- **Boost** - C++ utility libraries
